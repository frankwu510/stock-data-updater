## 🚀 使用方式

### 快速开始
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 测试模式
python src/stock_data_updater/main.py --test

# 3. 正式模式
python src/stock_data_updater/main.py --production

# 4. 批量下载
python download_all_stocks.py
```
