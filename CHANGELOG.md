# 变更日志

所有对项目的显著更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，

## [0.1.0] - 2026-04-13

### 新增

- 🎉 **初始版本发布**
- 📦 完整的 VS Code 项目结构
- 🔐 Baostock 登录/登出功能
- 📊 股票数据获取和更新功能
- 🔄 现有股票数据自动更新
- 🆕 新股数据文件创建
- 🧪 测试模式和正式模式
- 📁 自动数据目录管理
- 💾 CSV 格式数据保存（GBK 编码）
- 🎯 命令行参数支持
- 🔧 VS Code 调试配置
- 📝 完整的文档说明
- ✅ 单元测试框架

### 功能特性

- **数据获取**: 从 Baostock 获取历史股票数据
- **数据更新**: 自动检测并更新现有股票数据
- **文件管理**: 按股票代码自动创建和管理数据文件
- **错误处理**: 完善的异常处理和错误提示
- **编码支持**: 支持中文字符（GBK 编码）
- **请求控制**: 内置请求延迟，避免过于频繁访问

### 项目结构

- `src/stock_data_updater/` - 主要源代码
- `data/` - 正式数据目录
- `data_test/` - 测试数据目录
- `tests/` - 单元测试
- `.vscode/` - VS Code 配置文件
- `setup.py` - 项目安装脚本
- `requirements.txt` - 依赖包列表
- `README.md` - 项目说明
- `QUICK_START.md` - 快速开始指南

### 依赖包

- `pandas>=1.3.0` - 数据处理
- `baostock>=0.8.8` - 股票数据源

### 已知问题

- 新股检测功能为简化版本，需要手动维护股票代码列表
- 数据字段相对基础，可根据需要扩展
- 仅支持上海和深圳交易所股票
